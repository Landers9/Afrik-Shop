# README
***

## Versions Language
***
- Ruby 2.6.5
- Rails 5.2.5

## technology used
***
- devise for users 
- favorite function to add products to favorite
- Templates : Bootstrap and many librarie of javascript

## Execution procedure
***
> $ git clone https://github.com/Landers9/Afrik-Shop.git
> $ cd Afrik-Shop
> $ bundle install  
> $ rails db:create  
> $ rails db:migrate  
> $ rails db:seed  
> $ rails s
